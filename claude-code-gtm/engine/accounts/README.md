## How to Use

1. **Creating a card:** Copy `brain/account-card-template.md`, fill in all sections, save here
2. **After every touch:** Update the Touch Timeline table and Current Status section
3. **After a call:** Fill in Discovery Notes and Objections Raised
4. **After a win/loss:** Check the Brain Updates Triggered checkboxes
5. **After DNC:** Delete the card and note in state.md DNC list

**Rule:** If something happened at an account and the card isn't updated — it didn't happen for the engine.
