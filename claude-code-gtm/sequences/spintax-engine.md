## How to Load This Into Instantly

1. Open the sequence email in Instantly
2. Replace the static copy with the spintax version below
3. Instantly automatically selects one variant per send
4. Review analytics at 2-week mark â€” check which subject lines get highest open rate, which bodies get highest reply rate
5. Pause underperforming variants, scale winners

**Syntax rules:**
- `{A|B|C}` â€” Instantly picks one of A, B, or C each time
- Never spin `{{variables}}` â€” these are merge fields, not spin elements
- Never spin proof point numbers â€” specificity must be consistent
- Nested spintax is supported: `{He walked in|She walked into|{{proof_person}} arrived}` to include personalization inside spin
