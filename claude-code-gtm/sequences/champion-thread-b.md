# ChampionFollow_v1 — Thread B Champion Sequence
> Variant: `ChampionFollow_v1`
> Persona: Thread B — SDR Manager / Senior SDR / RevOps Lead / Sales Operations
> Trigger: Tier 1 account | Thread A active | Champion contact found with verified email
> Emails: 4 | Duration: 18 days
> Sending name: SDR name (always — peer-level contact works better here than Aaron)

---

## What This Sequence Is Doing

The champion is not the decision maker. They are the person who feels the problem most directly every single day.

The SDR Manager watching their team produce 8 meetings when the target is 25. The RevOps lead who knows the ICP definition is wrong but doesn't have the authority to rebuild it. The Senior SDR who's been told to "just send more emails."

This sequence speaks to that person — not to get them to buy, but to get them to care enough to bring SELLL into the conversation with the decision maker. We are building an internal advocate, not closing a deal.

**Critical rules:**
- Thread B ONLY starts if Thread A (decision maker) has received Email 1 with no positive reply after 5 days
- Thread B STOPS immediately the moment Thread A gets any positive reply (escalation, booking, even a question)
- Never pitch to the champion — champion emails are peer-level, empathetic, intelligence-sharing
- SDR's name and title on all emails — champion to champion is the right frame

## SDR Operating Notes

**Thread B timing rules (strictly enforced):**

| Thread A Status | Thread B Action |
|----------------|----------------|
| Thread A Email 1 sent, no reply after 5 days | START Thread B Email 1 |
| Thread A gets any positive reply | STOP Thread B immediately |
| Thread A gets a hard NO | STOP Thread B immediately |
| Thread A gets an OOO | Pause Thread B until Thread A resumes |
| Thread A gets a wrong person reply | Continue Thread B, update Thread A contact |

**Escalation trigger:**
- If Thread B champion forwards the thread to Thread A decision maker → immediately notify Aaron
- If Thread B champion replies positively → treat as HOT, escalate to Aaron
- If Thread B champion books a call → immediately pause Thread A and notify Aaron

**Finding the champion:**
- Look for: SDR Manager, Sales Development Manager, Head of SDR, RevOps Lead, Sales Operations Manager
- These are the roles that feel the ICP/targeting pain most acutely
- If the company has no one in these roles → Thread B goes to the most senior SDR, or CEO if < 30 employees
- If Thread B champion is the same person as Thread A decision maker → do not run Thread B

**Proof point for champion sequence:**
- Lead with the team-level outcome, not the revenue outcome
- "Less time on manual research" > "31 meetings in month 2"
- Champions want to see their team's daily experience change — the revenue impact is a secondary win
