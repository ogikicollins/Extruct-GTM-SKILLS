# Campaign Memory — SELLL.io
> Brain Layer: Institutional Knowledge | Updated: 2026-06-21
> Every campaign run is stored here. This is the pattern library that makes future campaigns smarter.

This file stores the learnings from every campaign SELLL runs — not the raw numbers (those are in revenue-reporting), but the *why behind the numbers*. What worked, what didn't, what surprised us, what buyer language we discovered, what hypotheses were confirmed.

---

## How to Use This File

**Before building a new campaign:** Read the most recent 3 campaign entries. Look for patterns in what hypothesis worked, what hook landed, what proof point converted.

**After a campaign closes:** Add a full debrief entry using the template below.

**Citation format for brain queries:**
> "Source: campaigns.md → [Campaign Name] → [Section]"
