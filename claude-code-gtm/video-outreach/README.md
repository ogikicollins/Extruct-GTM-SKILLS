# Video Outreach — SELLL.io
> Managed by: `video-outreach` skill | One script file per active Tier 1 contact
> Purpose: Per-contact video scripts for HeyGen AI video generation (Email 3).

All Tier 1 contacts receive a personalized video in Email 3. Scripts are generated automatically by `ai-personalization` skill (for reply_prob ≥ 70) or `video-outreach` skill (for reply_prob 35–69 Tier 1 contacts). Aaron records his HeyGen avatar once; the system personalizes at scale from that template.

---

## File Naming

`[company-slug]-[contact-slug]-video.md`

Examples:
- `prism-analytics-james-okafor-video.md`
- `dataflow-analytics-sarah-kim-video.md`

---

## Video Script File Format

```markdown
# Video Script — [First Name] [Last Name] at [Company]
> Generated by: [ai-personalization / video-outreach] skill
> Reply probability: [N]%
> Generated: [date]
> HeyGen video URL: [filled when rendered — written back to CSV v_loom_url]
> Status: [Queued / Rendering / Ready / Failed → text fallback]

## Script (45 seconds)

Hey [First Name] — Aaron here from SELLL.

Quick thing worth seeing if you're building pipeline at [Company Name].

We helped [Proof Company] go from [Proof Before] to [Proof After] in [Proof Timeframe]. [Client name] was [proof situation in one sentence].

If [Company Name] is working through something similar — worth 15 minutes.

[Calendar link]

## Personalization Variables Used
- first_name: [value]
- company_name: [value]
- proof_company: [value]
- proof_before: [value]
- proof_after: [value]
- proof_timeframe: [value]
- proof_situation: [value]
```

---

## Video Path by Reply Probability

| Reply Prob | Path | Who generates |
|-----------|------|--------------|
| ≥ 70 | HeyGen API (ai-personalization skill) | Automated via Claude API → HeyGen |
| 35–69 (Tier 1) | HeyGen API (video-outreach skill) | Automated — standard template script |
| < 35 (Tier 2) | No video | Standard Email 3 text only |
| > 85 (HOT, replied) | Optional manual Loom | Aaron records a personal Loom if contact is actively engaged |

**Important:** For contacts with reply_prob 35–69, the HeyGen script uses the standard template (no Claude-generated bespoke opener). The personalization is name, company, and proof point only — not a full bespoke script.
