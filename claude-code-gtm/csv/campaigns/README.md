# Campaign CSVs — SELLL.io
> Engine Layer: Activation | Updated: 2026-06-21
> Created by: `SELLL-layer-2` → Step 7
> Consumed by: Instantly (via `campaign-sending` skill)

One verified CSV per campaign run. Named by hypothesis and date.

---

## Naming Convention

`[hypothesis]-[YYYY-MM-DD]-verified.csv`

Examples:
- `H5-2026-06-21-verified.csv`
- `H1-2026-07-01-verified.csv`
- `H5-H1-compound-2026-07-15-verified.csv` (for compound signal campaigns)

---

## CSV Column Schema (v2.1 — 62 columns across 7 groups)

Every campaign CSV must have these columns before import to Instantly.
Generated by `SELLL-layer-2` Phase 5A. Do not manually edit column names.

The v_ prefix columns are Instantly merge fields — formatted for copy use.
The base columns (first_name, employee_count, etc.) are raw data for filtering and segmentation.
