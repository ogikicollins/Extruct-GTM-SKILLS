# LinkedIn Content Machine

The highest-converting touch in any B2B sequence is a DM **after the prospect already engaged with your content**. This skill runs the full loop: generate posts → publish → detect engagers → route them into a warm DM sequence that converts at 3–5x cold outreach.

It also builds SELLL.io's brand authority over time — so that when a cold email lands, the prospect already recognizes the name.

## How It Works

```
Week 1–4: Post 3x/week → build audience
After each post: run post-engagers → identify who liked/commented
Engager identified as ICP fit → warm DM sequence (not cold outreach)
Warm DM → 3–5x higher reply rate than cold email
Repeat weekly → compound audience + pipeline
```

## Post Framework

Post 3 times per week. Each post serves a different purpose in the pipeline:

| Day | Post Type | Purpose |
|-----|-----------|---------|
| Monday | **Case Study / Outcome Post** | Social proof — drives DMs from people in the same pain |
| Wednesday | **Insight / Data Post** | Authority — gets shared by GTM leaders, grows audience |
| Friday | **Framework / How-To Post** | Trust — positions SELLL as the expert, drives saves |

## Post Type 1: Case Study / Outcome Post

The highest-converting post type. Real numbers. Real company. Real before/after.

**Structure:**
```
Line 1 (HOOK): The result — specific, surprising, verifiable
Line 2: Set up the before state (what they were dealing with)
Lines 3–5: What changed (the system, not the magic)
Line 6: The after state
Line 7–8: The lesson / what this means for others in the same situation
CTA: One question that invites replies from people in the same pain
```

**Template:**
```
[Company] went from [X] to [Y] in [timeframe].

Before: [their situation in one sentence — the pain, the broken process]

We built:
→ [System component 1]
→ [System component 2]
→ [System component 3]

After: [specific outcome — numbers, not adjectives]

The lesson: [transferable insight — what this means for anyone in the same situation]

Question for you: [targeted question that invites replies from ICP prospects]
```

**Example for SELLL.io:**
```
Devolon's SDR team went from 35 daily activities to 200+ personalized conversations. Same headcount. 90 days.

Before: 40–50% of each SDR's day was list-building and manual research. Sequences were generic. Reply rates sat at 0.8%.

We built:
→ An AI-powered signal layer across 45+ data sources
→ Automated personalization triggered by real buying intent
→ Multi-channel orchestration (email + LinkedIn + ads in sync)

After: 200+ daily conversations running automatically. SDRs spend their time on warm replies and closing. Reply rate: 3.8%.

The lesson: the bottleneck in most outbound isn't the SDR — it's the system they're plugged into.

Question: what's the biggest time sink for your SDR team right now?
```

## Engager Detection and Warm DM Routing

After every post goes live, run `post-engagers` within 24–48 hours:

### Step 1: Identify engagers

Pull everyone who liked or commented on the post. For each person:
1. Check their role and company against ICP (title, company size, vertical)
2. Score against the 60-second ICP qualification check
3. Flag as ICP-fit engager if they pass

### Step 2: Classify engagement type

```
Comment (substantive) → WARM (they thought enough to write something)
Comment (emoji / "great post") → MILD WARM
Like only → MILD SIGNAL
Share → STRONG SIGNAL (they want their network to see this)
```

### Step 3: Route to warm DM sequence

For ICP-fit engagers who commented or shared:

**DM 1 — Sent within 24h of engagement:**
```
"Hey [Name] — saw your comment on my post about [topic]. [One sentence acknowledging what they said — show you read it].

Curious — is [the pain the post was about] something you're dealing with at [Company] right now?
```

**If they reply YES or show interest:**
```
DM 2: "Makes sense given where [Company] is at. We built a system that solves exactly that — [one-line outcome].

Worth 15 minutes to show you how it works? https://cal.com/collins-ogiki-x4fokk/30min"
```

**If they don't reply to DM 1 (send after 3 days):**
```
DM 2: "No worries if the timing isn't right — dropping a quick follow-up in case the first message got buried.

[Relevant case study / new proof point in one sentence].

Still curious if [the pain] is on your radar at [Company]."
```

### Step 4: Log warm leads separately

Track all warm LinkedIn leads in `claude-code-gtm/engine/linkedin-warm-leads.md`:

```
| Name | Company | Role | Post Engaged With | Engagement Type | DM Sent | Reply | Status |
```

These are separate from cold email pipeline. Track their conversion rate independently — it will be 3–5x higher.

## Workflow

1. Generate the week's 3 posts from the templates above
2. Customize with real proof points from context file Proof Library
3. Present to user for review and approval
4. User posts manually (or schedules via LinkedIn / Buffer)
5. 24–48h after each post: run `post-engagers` to identify ICP-fit engagers
6. Generate warm DM for each ICP-fit engager
7. User sends DMs (or approves for automated send)
8. Log all engagers and DM outcomes in linkedin-warm-leads.md
